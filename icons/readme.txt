Place your PWA icons here with the following filenames and sizes:

- icon-192.png  (192x192)
- icon-512.png  (512x512)

Best practice: create square PNGs with transparent background. You can generate them using an image editor or an online PWA icon generator.

If you don't have icons yet, you can generate simple placeholders using Node.js and the optional script `generate-placeholders.js` in the project root.
