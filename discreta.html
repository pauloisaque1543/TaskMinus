  1. 1. Use uma demonstração direta para mostrar que a soma de dois números inteiros ímpares é par.

    Solução (direta):

    Sejam m e n inteiros ímpares. Então existem inteiros k e ℓ tais que

    m = 2k + 1 e n = 2ℓ + 1.

    Somando:

    m + n = (2k+1) + (2ℓ+1) = 2(k+ℓ+1).

    Como k+ℓ+1 é inteiro, m+n é múltiplo de 2, logo é par.

    Tipo: demonstração direta.

  2. 2. Mostre que o quadrado de um número par é um número par (demonstração direta).

    Solução (direta):

    Seja n um inteiro par. Então existe inteiro k tal que n = 2k.

    Então

    n² = (2k)² = 4k² = 2(2k²).

    Como 2k² é inteiro, n² é múltiplo de 2, portanto par.

    Tipo: demonstração direta.

  3. 3. Demonstre que se m + n e n + p são números inteiros pares, em que m, n, p são inteiros, então m + p é par. Que tipo de demonstração você utilizou?

    Solução:

    Suponha que m+n = 2a e n+p = 2b para alguns inteiros a,b.

    Subtraindo as duas igualdades:

    (m+n) - (n+p) = 2a - 2b ⇒ m - p = 2(a-b).

    Assim m-p é par; escreva m-p = 2c para algum inteiro c. Somando 2p aos dois lados:

    m+p = 2c + 2p = 2(c+p), que é par.

    Tipo: demonstração direta (manipulação algébrica).

  4. 4. Use uma demonstração direta para mostrar que todo número inteiro ímpar é a diferença de dois quadrados.

    Solução (direta):

    Seja n um inteiro ímpar. Então n = 2k + 1 para algum inteiro k.

    Calcule a diferença:

    (k+1)² - k² = (k² + 2k + 1) - k² = 2k + 1 = n.

    Logo n é a diferença dos quadrados de k+1 e k.

    Tipo: demonstração direta.

  5. 5. Use uma demonstração por contradição para provar que a soma de um número irracional e um racional é irracional.

    Solução (contradição):

    Seja x irracional e q racional. Queremos mostrar que x + q é irracional.

    Suponha, para contradizer, que x + q seja racional. Então existe r ∈ ℚ tal que x + q = r. Assim

    x = r - q.

    Mas a diferença de dois racionais é racional, logo x seria racional — contradição com a hipótese de que x é irracional.

    Portanto x + q é irracional.

    Tipo: demonstração por contradição.

  6. 6. Demonstre ou contrarie que o produto de dois números irracionais é irracional.

    Resposta:

    A afirmação “o produto de dois irracionais é sempre irracional” é falsa.

    Contraexemplo: √2 e √2 são irracionais, mas √2 · √2 = 2, que é racional.

    Observação: também há casos em que o produto é irracional (por exemplo, √2 · √3 = √6, que é irracional). Portanto o produto pode ser racional ou irracional dependendo dos fatores.

    Tipo: contraexemplo (refutação).

  7. 7. Demonstre que se x é irracional, então 1/x é irracional (supondo x ≠ 0).

    Solução (contradição):

    Suponha x irracional e x ≠ 0. Suponha, para contradizer, que 1/x seja racional, isto é, 1/x = r ∈ ℚ.

    Então x = 1/r. Mas o inverso de um racional não nulo é racional, logo x seria racional — contradição.

    Portanto 1/x é irracional.

    Tipo: demonstração por contradição.

  8. 8. Use uma demonstração por contraposição para mostrar que se x + y ≥ 2, em que x e y são reais, então x ≥ 1 ou y ≥ 1.

    Solução (contrapositivo):

    Proposição original: (x+y ≥ 2) ⇒ (x ≥ 1 ∨ y ≥ 1).

    O contrapositivo é: (x < 1 ∧ y < 1) ⇒ (x+y < 2). Se o contrapositivo é verdadeiro, a proposição original também é.

    Se x < 1 e y < 1, então x+y < 1 + 1 = 2. Assim o contrapositivo é verdadeiro, logo a proposição original vale.

    Tipo: demonstração por contraposição.

  9. 9. Mostre que se n é um número inteiro e n³ + 5 é ímpar, então n é par, usando:

    (a) Uma demonstração por contraposição.

    Contrapositivo: se n é ímpar, então n³ + 5 é par.

    Se n é ímpar, n = 2k+1. Então

    n³ = (2k+1)³ = 8k³ + 12k² + 6k + 1, que é ímpar (da forma 2·(algo) + 1).

    Como 5 é ímpar, n³ + 5 = ímpar + ímpar = par. Logo o contrapositivo é verdadeiro, e a implicação original vale.

    (b) Uma demonstração por contradição.

    Suponha que n³ + 5 seja ímpar e, para contradizer, que n seja ímpar.

    Como acima, se n é ímpar então n³ é ímpar, e então n³ + 5 = ímpar + ímpar = par — contradição com a hipótese. Logo n não pode ser ímpar, portanto n é par.

    Tipos: (a) contraposição; (b) contradição.

  10. 10. Demonstre a proposição P(0), em que P(n) é a proposição “Se n é um número inteiro positivo maior que 1, então n² > n”. Qual tipo de demonstração você utilizou?

    Observação e resposta:

    A proposição P(n) está formulada para inteiros positivos maiores que 1, portanto P(0) não é aplicável (0 não satisfaz a hipótese “> 1”).

    Se a intenção era checar a base de uma indução, normalmente verificaríamos P(2): para n = 2, 2² = 4 > 2, verdadeiro.

    Para verificar um caso específico (como n=2) usa-se demonstração direta (checagem do caso). Se precisassem provar para todo n ≥ 2, usar-se-ia indução com base e passo indutivo.

    Tipo: verificação direta do caso base (ou observação sobre domínio da proposição).

  11. 11. Assuma P(n) como a proposição “Se a e b são números reais positivos, então (a + b)n ≥ an + bn”. Comprove que P(1) é verdadeira. Qual tipo de demonstração você utilizou?

    Solução:

    Para n = 1, temos (a+b)1 = a + b. Logo

    a + b ≥ a + b, que é verdadeiro (igualdade).

    Tipo: verificação direta (checar o caso n = 1).

  12. 12. Mostre que pelo menos 10 de quaisquer 64 dias escolhidos devem cair no mesmo dia da semana.

    Solução (princípio da casa dos pombos):

    Existem 7 dias da semana (7 "caixas"). Se colocarmos 64 dias em 7 caixas, uma das caixas terá no mínimo

    ⌈64 / 7⌉ = ⌈9,142...⌉ = 10 elementos.

    Portanto, pelo menos 10 dos 64 dias cairão no mesmo dia da semana.

    Tipo: princípio da casa dos pombos (pigeonhole).

  13. 13. Use uma demonstração por contradição para mostrar que não há um número racional r para que r³ + r + 1 = 0.

    Solução (usando o Teorema das raízes racionais):

    Considere o polinômio f(t) = t³ + t + 1. Pelas possíveis raízes racionais (teorema das raízes racionais), qualquer raiz racional p/q em forma reduzida deve satisfazer p | 1 e q | 1, isto é, os únicos candidatos racionais são ±1.

    Avalie:

    f(1) = 1 + 1 + 1 = 3 ≠ 0

    f(-1) = -1 + (-1) + 1 = -1 ≠ 0

    Como nem 1 nem -1 são raízes, o polinômio não tem raízes racionais.

    Isso prova que não existe r ∈ ℚ com r³ + r + 1 = 0.

    Tipo: contradição/teorema das raízes racionais (ou prova por contradição equivalente usando divisibilidade).

  14. 14. Demonstre que se n é um número inteiro positivo, então n é ímpar se e somente se 5n + 6 for ímpar.

    Solução (redução módulo 2):

    Trabalhe módulo 2. Como 5 ≡ 1 (mod 2) e 6 ≡ 0 (mod 2), temos:

    5n + 6 ≡ 1·n + 0 ≡ n (mod 2).

    Logo 5n + 6 tem a mesma paridade que n. Assim 5n+6 é ímpar ⇔ n é ímpar.

    Tipo: congruência módulo 2 (prova direta).

  15. 15. Demonstre ou contrarie que se m e n são inteiros, tal que mn = 1, então ou m = 1 e n = 1 ou m = −1 e n = −1.

    Solução:

    Se m e n são inteiros e m·n = 1, então m é divisor de 1. Os divisores inteiros de 1 são apenas 1 e -1.

    Logo m = 1 ou m = -1. Se m = 1, então n = 1. Se m = -1, então n = -1.

    Portanto as únicas possibilidades inteiras são (1,1) ou (-1,-1).

    Tipo: argumento direto sobre divisores inteiros.

  16. 16. Mostre que estas proposições sobre o número inteiro x são equivalentes: (i) 3x + 2 é par, (ii) x + 5 é ímpar, (iii) x² é par.

    Solução (mostrar que todas equivalem a “x é par”):

    (i) ⇒ (x é par): Se 3x + 2 é par, então 3x é par (subtraindo 2). Como 3 é ímpar, isso só ocorre se x for par.

    (x é par) ⇒ (ii): Se x é par então x + 5 = par + ímpar = ímpar. Portanto (i) ⇒ (ii).

    (ii) ⇒ (x é par): Se x + 5 é ímpar então x é par (ímpar − 5(ímpar) = par).

    (x é par) ⇒ (iii): Se x é par então x² também é par.

    Assim todos implicam “x é par”, logo (i), (ii) e (iii) são equivalentes.

    Tipo: sequências de implicações (reduzir todas à condição “x é par”).

  17. 17. Mostre que estas proposições sobre o número real x são equivalentes: (i) x é irracional, (ii) 3x + 2 é irracional, e (iii) x/2 é irracional.

    Solução:

    (i) ⇒ (ii): Se x é irracional então 3x é irracional (multiplicar por 3, racional não nulo, preserva irracionalidade). Somar 2 (racional) a um irracional gera um irracional — logo 3x + 2 é irracional.

    (ii) ⇒ (i): Se 3x + 2 fosse irracional e x racional, então 3x + 2 seria racional (contradição). Assim x é irracional.

    (i) ⇔ (iii): Multiplicar ou dividir por 2 (racional não nulo) preserva irracionalidade: x irracional ⇔ x/2 irracional.

    Portanto (i), (ii) e (iii) são equivalentes.

    Tipo: argumentos diretos e contradição quando necessário.

  18. 18. Os passos abaixo para encontrar as soluções de √(x + 3) = 3 − x estão corretos?

    (1) √(x + 3) = 3 − x   (dado)
(2) x + 3 = (3 − x)²   (elevar ambos os lados ao quadrado)
(3) x + 3 = 9 − 6x + x²
(4) 0 = x² − 7x + 6
(5) 0 = (x − 1)(x − 6)   ⇒ soluções formais x = 1 ou x = 6

    Análise e conclusão:

    Os passos algébricos estão corretos (elevar ao quadrado gerou a equação quadrática). Porém é essencial verificar as soluções no enunciado original porque elevar ao quadrado pode introduzir raízes extranhas.

    Verificação:

    - Para x = 1: √(1+3) = √4 = 2 e 3 − 1 = 2 ⇒ funciona.

    - Para x = 6: √(6+3) = √9 = 3, mas 3 − 6 = −3. Como a raiz principal é não-negativa, 3 ≠ −3, logo x = 6 é extraneous.

    Conclusão: somente x = 1 é solução válida.

    Tipo: verificação de soluções após elevar ao quadrado (checar extraneous roots).

  19. 19. Comprove que pelo menos um dos números reais a1, a2, ..., an é maior que ou igual ao valor da média desses números. Que tipo de demonstração você utilizou?

    Solução (contradição):

    Seja a média m = (a₁ + a₂ + ... + aₙ) / n. Suponha, por contradição, que todos os aᵢ < m.

    Somando essas desigualdades obtemos a₁ + ... + aₙ < n·m, mas por definição n·m = a₁ + ... + aₙ, contradição.

    Logo existe pelo menos um aᵢ tal que aᵢ ≥ m.

    Tipo: demonstração por contradição.

  20. 20. Comprove que se n é um número inteiro, estas quatro proposições são equivalentes: (i) n é par, (ii) n + 1 é ímpar, (iii) 3n + 1 é ímpar, (iv) 3n é par.

    Solução (cadeia de implicações):

    (i) ⇒ (ii): Se n é par então n = 2k e n+1 = 2k+1 é ímpar.

    (ii) ⇒ (iii): Se n+1 é ímpar então n é par. Se n é par, 3n é par e 3n+1 é ímpar.

    (iii) ⇒ (iv): Se 3n+1 é ímpar então 3n é par (subtraia 1).

    (iv) ⇒ (i): Se 3n é par e 3 é ímpar, isso só acontece se n for par (produto de ímpar por ímpar é ímpar).

    Assim (i)⇒(ii)⇒(iii)⇒(iv)⇒(i), o que mostra que todas são equivalentes.

    Tipo: demonstração por implicações encadeadas (equivalência).
